# 法律案件系统用户手册

> 版本：`2026-03-27`
> 说明：本文只描述当前仓库已经实现、且对用户真实开放的能力。

## 系统定位

系统当前服务四类角色：

- `super_admin`：平台级租户与用户治理，只用 Web。
- `tenant_admin`：机构管理员，可用 Web 与小程序。
- `lawyer`：律师，可用 Web 与小程序。
- `client`：当事人，当前业务侧以小程序为主。

## 角色与权限

| 角色 | 主使用端 | 当前可做什么 | 当前限制 |
| --- | --- | --- | --- |
| `super_admin` | Web | 看平台总览、租户、全局用户，维护租户状态与 AI 预算 | 不参与案件业务 |
| `tenant_admin` | Web + 小程序 | 管案件、当事人、律师审批、邀请、分析管理 | 只限本租户 |
| `lawyer` | Web + 小程序 | 建案、查案、上传、改案件状态、看 AI 结果 | 不能做平台治理 |
| 个人工作区律师 | Web + 小程序 | 使用案件相关能力 | 只保留案件相关导航 |
| `client` | 小程序 | 看本人案件、上传材料、补充说明、看摘要与下载本人可见内容 | Web 业务工作台不可用，不能发起 AI |

## 登录与状态说明

### Web

支持：

1. 账号密码登录
2. 短信验证码登录
3. 微信扫码登录

说明：

- 账号主键是手机号。
- 登录页会先调用前置判断接口（`/auth/login-advice`），给出是否必填 `tenant_code`、是否待审批、推荐登录端。
- 同手机号命中多个租户时，需要补 `tenant_code`。
- 当事人登录 Web 后会被引导到“小程序使用提示页”。
- 支持排查统一三问：当前角色、当前端、是否首登或刚被邀请。

### 小程序

支持：

1. 微信一键登录
2. 短信验证码登录
3. 账号密码登录

并支持：

- 案件邀请进入后自动绑案
- 律师邀请进入后走待审批流程
- Web 扫码登录确认

### 待审批与首登改密

- 新加入机构的律师可能先进入“待审批”页。
- 首次登录或被要求重置密码时，会强制进入改密页。

## Web 端说明

### 默认落地页

| 角色 | 默认页面 |
| --- | --- |
| `super_admin` | 平台总览 |
| `tenant_admin` | 概览 |
| `lawyer` | 概览 |
| 个人工作区律师 | 案件管理 |
| `client` | 小程序使用提示页 |

### 机构管理员 / 律师

当前可用页面：

- 概览
- 案件管理
- 案件详情
- 当事人管理
- 律师管理（仅机构管理员）
- 文档解析页

当前行为限制：

- Web `/analysis` 仍重定向到概览。
- Web “法律分析 / 真伪核验”路由当前重定向到文档解析页。
- Web 案件邀请入口不要视为稳定主入口，实际推荐在小程序案件详情里生成邀请。

### 超级管理员

当前可用页面：

- 平台总览
- 租户管理
- 全局用户总览

当前定位：

- 偏治理、审计、预算配置
- 不直接参与案件

## 小程序说明

### 机构管理员 / 律师

#### 底部菜单

- 机构管理员：概览、案件、当事人、律师、我的
- 普通律师：概览、案件、当事人、我的
- 个人工作区律师：案件、我的

#### 当前可做的事

- 查看概览与案件列表
- 新建案件
- 查看案件详情、时间线、补充说明、律师备注
- 上传、预览、下载、删除材料
- 生成当事人邀请路径
- 机构管理员进入分析管理页

### 当事人

#### 进入方式

推荐通过案件邀请进入，登录后自动绑定案件。

#### 底部菜单

- 案件
- 我的

#### 当前可做的事

- 查看本人案件
- 查看案件进度与可见时间线
- 上传材料
- 填写补充说明
- 下载本人可见文件与最新报告
- 查看 AI 摘要

#### 看不到的内容

- 律师内部备注
- 证伪详情
- 其他人的案件与材料

## 常见问题

### 为什么当事人不能正常使用 Web 端？

因为当前产品边界就是“当事人业务能力在小程序开放，Web 只做提示页”。

### 为什么要求填写租户编码？

因为同一手机号可以存在于多个租户，系统需要 `tenant_code` 定位账号。登录页会基于前置判断接口提前提示，不再依赖报错后补填。

### 为什么我能登录但进不了工作台？

常见原因：

- 账号待审批
- 首次登录必须先改密
- 当前角色不允许进入该端的业务页

### 为什么个人工作区律师看不到概览、当事人、律师管理？

这是当前实现限制，不是异常；个人工作区只保留案件相关入口。
